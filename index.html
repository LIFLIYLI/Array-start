数组的重要属性

forEach的用法

forEach是数组的遍历
var arr = [1,2,3]
arr.forEach(function(x,y){
console.log(x)
//函数中的参数x表示arr中的value
console.log(y)
//y表示arr中的key
})
上诉函数简写为：
arr.forEach((x,y)=>{
console.log(x)
//函数中的参数x表示arr中的value
console.log(y)
//y表示arr中的key
})

map的用法

map的用法和forEach差不多，但是forEach没有返回值，map有返回值
map有2个重要的属性
1：Map.prototype.constructor
//返回一个函数，这个不了解
2:Map.prototype.size
//返回Map对象的键/值对的数量
他还有一些方法，如：
1：Map.prototype.set(key, value)
//设置Map对象中键的值。返回该Map对象。
例：
var myMap = new Map();
myMap.set(0, "zero");
myMap.set(1, "one");
上述代码则会设置并返回一个Map对象//Map(2) {0 => "zero", 1 => "one"}
2：Map.prototype.delete(key)
如果 Map 对象中存在该元素，则移除它并返回 true；否则如果该元素不存在则返回 false
例：
myMap.delete(0)
删除了第0项，返回新的对象//Map(1) {1 => "one"}
3：Map.prototype.get(key)
返回键对应的值，如果不存在，则返回undefined。
例：
myMap.get(0)
返回一个undefined,如果是myMap.get(1),则会返回字符串"one"

filter的用法

filter相当于滤网，根据条件进行筛选
例：
arr=[1,2,3,4]
arr.filter(function(value,key){
return value>=2
})
得到的结果为[2,3,4],其中函数中的第一个参数为数组的value,第二个为key

reduce的用法

reduce是相当于累计器
例：
arr.reduce(function(x,y){
return x+y
},0)
得到的结果为10，（1+2+3+4）
步骤是数组两两顺序相加，第一次是1+2，为3
其中参数x相当于每次前一个两两相加之和，即为3
y相当于两两相加之后的那个数，即为3
第二次是3+3等于6，这一次x就为6，y就为4
最后6+4，结果为10
